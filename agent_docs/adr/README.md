# ADR — журнал значимых решений

Каждое значимое решение — отдельный файл.

## Конвенция

- Имя: `YYYY-MM-DD-HHMM-short-title.md`.
- Время указывать локальное для проекта/сессии.
- Не вести ручную сквозную нумерацию: она конфликтует при параллельной работе агентов.
- Шаблон: `agent_docs/templates/adr.md`.
- Принятое ADR не редактировать без явной необходимости — создавать новое со статусом «Заменяет ...» и ссылкой на старое.
- Не обновлять этот README при каждом новом ADR.
- Глубину просмотра ADR агент определяет сам по задаче: учитывать риск, затронутые файлы, связанные решения и результаты поиска по ключевым словам.
