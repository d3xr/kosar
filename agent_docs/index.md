# Навигация по документам

Короткая карта для агентов. Читать только релевантные файлы.

## Контекст

- `AGENTS.md` — правила работы, структура проекта, команды.
- `README.md` — человеческий вход в проект.
- `agent_docs/glossary.md` — словарь железа, протоколов и кодовых названий.

## Архитектура

- `agent_docs/architecture.md` — текущая архитектура ESP32, ELRS и hoverboard controller.
- `docs/garage-runbook.md` — практический runbook по стенду и железу.
- `docs/telemetry-contract.md` — контракт телеметрии.

## Журналы

- `agent_docs/adr/` — значимые решения, по одному файлу на решение.
- `agent_docs/development-history/` — нетривиальные итерации, по одному файлу на итерацию.

## Гайды

- `agent_docs/guides/dod.md` — definition of done.
- `agent_docs/guides/environment-setup.md` — локальное окружение и секреты.
- `agent_docs/guides/logging.md` — логи для scripts/integrations.
- `agent_docs/guides/atomic-documents.md` — атомарные журналы.
- `agent_docs/guides/archiving-and-temp.md` — временные файлы и архив.

## Шаблоны

- `agent_docs/templates/architecture.md`
- `agent_docs/templates/adr.md`
- `agent_docs/templates/development-history.md`
