# Критерии завершенности (DoD)

## Минимальный DoD

- [ ] Результат соответствует требованиям задачи
- [ ] Каждое изменение прослеживается к исходному запросу (нет «попутных улучшений»)
- [ ] Тесты проходят; добавлены новые при необходимости — если применимо к коду (иначе N/A)
- [ ] Документация обновлена; для нетривиальных изменений — отдельная запись в `agent_docs/development-history/`

## Дополнительно по типам задач (если применимо)

### Новая фича / артефакт

- [ ] Обновлён CHANGELOG (если ведётся)
- [ ] Обновлена документация пользователя

### Багфикс

- [ ] Выявлена root cause
- [ ] Добавлен регрессионный тест (если применимо)

### Рефакторинг / реструктуризация

- [ ] Функциональность/смысл не изменены (подтверждено тестами или ревью)
- [ ] ADR создан при значимых изменениях

### Документация / контент

- [ ] Информация актуальна
- [ ] Примеры добавлены (если применимо)
- [ ] Ссылки работают
