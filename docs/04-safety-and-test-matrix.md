# Safety Gates And Test Matrix

Главное правило MVP: нож не "не используется", а физически заблокирован. RC-ходовая тестируется отдельно от режущего узла.

## Test Matrix

| Gate | Проверка | PASS-критерий |
|---|---|---|
| Диагностика до прошивки | Фото платы, MCU, SWD, фаз, Hall, батареи, MOSFET, разъемов | Gen1/Gen2 подтвержден, воды/горения нет |
| Родной гироскутер | Оба колеса до разборки крутятся штатно | Нет хруста, ошибок, просадки батареи |
| Питание платы | Только родная 10S батарея или CC-блок 36-42V | Не Greenworks 48V |
| ST-Link | Только `GND/SWDIO/SWCLK`, плата питается отдельно | `3.3V` от ST-Link не подключен |
| Unlock / flash | Кнопка питания удерживается весь процесс | Прошивка не прерывается |
| CC smoke test | Первый power-on после прошивки с ограничением 1-2A | Нет мгновенного current limit, дыма, перегрева |
| Hall / фазы | Цвет-в-цвет, минимальная команда на вывешенных колесах | Плавный старт без дерганья |
| RX / ESP32 | Сигнал, земля общая, питание проверено мультиметром | На RX/ESP32 не попадает 12/15V |
| Failsafe | Потеря пульта, RX power, signal wire | Команды становятся нулем или моторы уходят в timeout-stop |
| E-stop | Физический разрыв силового питания привода | Доступен снаружи и срабатывает сразу |
| Подставки | Газ, реверс, поворот, торможение | Направления понятны, самодвижения нет |
| Greenworks mount | Установка без ножа и без батареи ножа | Крепеж, клиренс, кабели и центр тяжести норм |
| Первый ground test | Ровная площадка, малая скорость | Стоп, failsafe и механика выдерживают |

## Critical Risks

- Greenworks 48V fully charged около 54V; не подключать к generic hoverboard mainboard.
- Sideboard cable может иметь 12/15V на красном проводе; приемник и ESP32 питать от проверенного DC-DC.
- На некоторых платах цвета проводов могут не совпадать с ожидаемыми; GND проверять мультиметром.
- MCU unlock может необратимо стереть стоковую прошивку.
- Нож Greenworks нельзя включать через обычное реле; для будущего этапа нужен отдельный safety design: контактор/MOSFET, e-stop, failsafe, interlock.

## WARN Risks

- PPM и длинные сигнальные провода чувствительны к шуму от моторов.
- Skid-steer на траве дает пики тока и рвет газон; pivot-turn запрещен на первых тестах.
- CC-блок 1-2A годится для smoke test платы, но не доказывает тягу моторов под нагрузкой.
- 3D-печать годится для корпусов и шаблонов, но не для силовых креплений осей.
- Влага, трава и вибрация требуют герметичного корпуса и разгрузки кабелей.

## Stop Criteria

Тест останавливается сразу, если:

- мотор дергается/гудит вместо плавного вращения;
- плата или MOSFET быстро греются;
- failsafe не останавливает моторы;
- есть неясность по земле/питанию receiver/ESP32;
- крепление оси имеет люфт или проворот;
- нож физически не отключен.
